# Kafview
Simple Kafka Message UI, support Avro Schema.

## Init config
http://localhost:9000 

## Add Config
http://localhost:9000/config

## Check Message
http://localhost:9000/messages

## Check Specific Topic Message
http://localhost:9000/messages?name=<topicName>&timeInSec=<timeInSec>
